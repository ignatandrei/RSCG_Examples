Original Readme

note

LightweightObjectMapper

1. Intro

A out of the box object mapper library based on SourceGenerator. SourceGenerator

2. Features

  • Out of the box, no pre-configuration required
  • No runtime library reference
  • Observable mapping code
  • No Reflection
  • EmitNo emit or other dynamic generations
  • Implementation based on extension methods, non intrusive target type

Note!!!
