About

Turns a folder tree into strongly-typed path constants at compile time — access file/folder paths as typed properties instead of magic strings.How to use 1. Declare which folder to scan in .csproj:```xml <ItemGroup> <PackageReference Include="TypedPaths.Generator" Version="1.3.0" /> </ItemGroup> <ItemGroup> <TypedPathsFolder Include="a" ClassName="Src" /> </ItemGroup>``` 2. Access generated typed path constants:```charp Console.WriteLine(TypedPaths.Src.Andrei.Value); // strongly-typed path to the "Andrei" entry under folder "a" (e.g. "a/Andrei" or "a\Andrei" depending on OS) ```You can use for Eliminating magic path strings for file system operations — folder/file paths become compile-time constants with IntelliSense and refactoring safety.
